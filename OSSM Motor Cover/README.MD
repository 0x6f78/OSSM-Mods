### Requirements

To able to use this motor cover, you need to have the latest version of the OSSM Motor Ring for the 57AIM30.

---
### BOM (Bill of Materials)

```
3x M4x16 Socket Head Cap Screw
4x M5x12 Socket Head Cap Screw
```
*That's it, nothing else needed :)*

---
### Printing (Files & Settings)

All print files are in the 3MF folder. They are not preconfigured, so make sure you set it right as per the settings below.

```
0.2mm layer height
Standard strength (5 wall) profile
15% gyroid infill
Support: Tree Slim + 10% Threshold angle (only for the cover ring)
```
